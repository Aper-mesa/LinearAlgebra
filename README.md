## 项目介绍
    本项目可用于繁琐的线代相关计算，追求实用性和精确性。
    Java版本：20
### 1. 分数类
__此类用于封装所有数据，包括整数，作用是以分数形式打印实数，以及数据的无损存储。此类将被应用于接下来的所有工具类。__
- 支持用户输入整数、小数以及指定格式的分数
- 支持整数、小数以及分数之间的四则运算
- 支持数据无损无精度问题
- 支持小数以分数形式输出
### 2. 行列式计算器
__此类将接收用户输入的行列式并给出行列式的值。__
### 3. 矩阵计算器
__此类用于矩阵相关的各种运算。__
- 加法
- 减法
- 乘法
- 方幂
- 数乘
- 求秩
- 求逆
- 转置
